# Human-Activity-Recognition-System-using-Hidden-Markov-Model
The Human Activity Recognition (HAR) system developed in this project addresses the challenge of identifying physical states from continuous, high-frequency motion data. By utilizing the WISDM dataset, the system was built on a foundation of real-world accelerometer readings. A critical preprocessing phase involved balancing the dataset to ensure that the "Sitting" activity was not overshadowed by more frequent "Walking" and "Jogging" entries, followed by the creation of temporal windows. Unlike traditional classifiers that treat each data point in isolation, the Hidden Markov Model (HMM) approach interprets these windows as a sequence of hidden states, allowing the model to capture the transition probabilities inherent in human motion. To further enhance the feature space, the tri-axial raw data was augmented with Euclidean Magnitude, providing a rotation-invariant measure of movement intensity that significantly boosted the model's discriminative power. The technical core of the system is a bank of Gaussian HMMs, where a dedicated model is trained for each specific activity. During inference, the system calculates the loglikelihood of a new sensor sequence against all trained models, selecting the activity corresponding to the highest probability. A major practical achievement of this project was the implementation of a Universal Data Inference Engine within a Streamlit web application. This engine proactively identifies and corrects discrepancies in sensor units and formats—such as converting G-force to or applying a gravity offset to linear acceleration data. This ensures that the model remains accurate regardless of the mobile application used for data collection. The final implementation achieved a robust accuracy of 97.02%, demonstrating that probabilistic sequential modeling is an efficient
and highly reliable solution for mobile-based activity recognition.
